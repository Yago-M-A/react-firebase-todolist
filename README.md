## React ToDoList
Projeto desenvolvido com Typescript, ReactJS e Firebase, com a proposta de ser uma aplicação de criar uma lista de tarefas, podendo criar, editar e deletar.

- [Live Preview](https://pokemonsearchapi.netlify.app/)
